# @xmpp/core

Installable xMPP SDK for paid-tool routing on Stellar.

Use this package when you want:

- a gateway client for an existing xMPP deployment
- in-process `xmppFetch` helpers through `@xmpp/core/local`
- workflow estimation and route explanation against the xMPP router model
- bootstrap utilities through the `xmpp-demo` CLI

Project documentation lives at the repo root:

- https://github.com/Vinaystwt/xMPP
- https://github.com/Vinaystwt/xMPP/blob/main/docs/sdk.md
